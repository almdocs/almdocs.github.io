Страница администратора
=======================

На странице администратора можно сделать ещё больше удивительных вещей! 
Ниже представлены некоторые из них:

* Удалять экзамены
* Редактировать данные о преподавателях
* Удалять и добавлять преподавателей

Инструкции по применению и использованию тех или иных функций представлены ниже!

Экзамены
------------------

Для просмотра этого раздела, необходимо нажать на пункт "Экзамены" в верхнем меню.
На данный момент это раздел по умолчанию.

Удаление
~~~~~~~~~~~~~~~~~~
  #. Выберите экзамен
  #. Нажмите кнопку "удалить данный экзамен"

Преподаватели
------------------------------

Для просмотра этого раздела, необходимо нажать на пункт "Преподаватели" в верхнем меню.

Добавление
~~~~~~~~~~~~~~~~~~
  #. Перейдите на вкладку "Добавить преподавателя"
  #. Заполните Имя, Фамилию, Отчество, Логин и Пароль для нового преподавателя
  #. Нажмите кнопку "добавить преподавателя"

Редактирование
~~~~~~~~~~~~~~~~~~
  #. Выберите интересующего Вас преподавателя
  #. Можете отредактировать любые данные о преподавателе
  #. Нажмите кнопку "обновить данные"

Удаление
~~~~~~~~~~~~~~~~~~
  #. Выберите преподавателя
  #. Нажмите кнопку "удалить преподавателя"
