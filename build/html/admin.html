Страница администратора¶

На странице администратора можно сделать ещё больше удивительных вещей! Ниже представлены некоторые из них:

  • Удалять экзамены
  • Редактировать данные о преподавателях
  • Удалять и добавлять преподавателей

Инструкции по применению и использованию тех или иных функций представлены ниже!

Экзамены¶

Для просмотра этого раздела, необходимо нажать на пункт «Экзамены» в верхнем меню. На данный момент это раздел по умолчанию.

Удаление¶

  1. Выберите экзамен
  2. Нажмите кнопку «удалить данный экзамен»

Преподаватели¶

Для просмотра этого раздела, необходимо нажать на пункт «Преподаватели» в верхнем меню.

Добавление¶

  1. Перейдите на вкладку «Добавить преподавателя»
  2. Заполните Имя, Фамилию, Отчество, Логин и Пароль для нового преподавателя
  3. Нажмите кнопку «добавить преподавателя»

Редактирование¶

  1. Выберите интересующего Вас преподавателя
  2. Можете отредактировать любые данные о преподавателе
  3. Нажмите кнопку «обновить данные»

Удаление¶

  1. Выберите преподавателя
  2. Нажмите кнопку «удалить преподавателя»
